Ke spuštění dev serveru nebo sestavení projektu je potřeba Node a npm: https://nodejs.org/
Produkční verze je statická stránka, lze provozovat téměř kdekoliv.

1) Pomocí npm nainstalujte depencence uvedené v package.json příkazem `npm install`
2) Pro sestavení spusťte `npm run build`
3) Produkční verze projektu je nyní ve složce /dist připravená k nasazení na libovolný web server

Alternativně je poslední verze v provozu na https://fh.ovi.moe/
