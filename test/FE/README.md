# Focus Hell
Projekt ITU 2019 – Hra Pexeso.

Ke spuštění dev serveru nebo sestavení je potřeba Node a npm.

## Příprava projektu
```
npm install
```

### Vývojový server s hot-reload funkcí
```
npm run serve
```

### Sestavení pro nasazení do provozu
```
npm run build
```

### Kontrola linterem
```
npm run lint
```
